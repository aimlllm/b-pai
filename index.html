考试概览

考试形式

  • 在 PSI 考试中心电脑上进行
  • 闭卷考试，不得携带任何资料
  • 115 道单选题，每题 4 个选项
  • 考试时间 210 分钟（3.5 小时）
  • 需要达到 72% 正确率方可通过
  • 猜题不扣分，建议全部作答

两部分考试

  • 法律与商业考试 (Law & Business)：所有申请人必须通过
  • 行业专业考试 (Trade Exam)：B 牌 General Building 专业考试
  • 两项考试均需通过才能获得执照

考试费用与报名

  • 通过 CSLB 官网申请
  • 需要至少 4 年相关工作经验
  • 或相关学历 + 工作经验组合
  • 申请费用约 $450
  • 需提供指纹及背景调查

考试策略

  • 平均每题约 1.8 分钟
  • 先做有把握的题目
  • 标记不确定的题目回头检查
  • 选择"最佳答案"而非"正确答案"
  • 注意题目中的关键词：最小、最大、至少等
  • 不确定时大胆猜测，不要留空

涵盖执照申请、法律商业、建筑规范、施工知识、电气、管道暖通、安全等全部考试内容

加州B牌承包商执照完全指南 / Complete Guide to California B Contractor License

什么是加州B牌承包商执照？ What is a California Class B Contractor License?

加州B牌承包商执照（California Class B General Building Contractor License）是由加州承包商执照委员会（CSLB, California Contractors State License Board）颁发的专业执照。B类执照持有者可以承接涉及至少两个不相关建筑工种的结构性建筑项目总包合同，包括住宅和商业建筑的新建、改建和翻修。这是加州最常见的承包商执照分类之一。根据加州法律（Business and Professions Code Section 7000 et seq.），任何金额达到$1,000或以上的建筑项目都必须由持照承包商执行。

申请流程8步骤 / 8-Step Application Process

获得加州B牌承包商执照的完整流程通常需要6-9个月，总费用约$1,500-$3,000。以下是详细步骤：

  1. 选择执照分类和资格认证人 — 选择B类 General Building 分类，确定资格认证人(Qualifier)，可以是RME（负责管理员工）或RMO（负责管理官员）。
  2. 记录工作经验 — 提交由雇主签署的工作经验证明表，要求过去10年内至少4年journeyman级别工作经验。
  3. 提交申请 — 在CSLB官网在线完成或下载纸质申请，申请费$450。
  4. 指纹与背景调查 — 在授权地点完成Live Scan指纹采集（约$49-$59）。
  5. 等待处理 — CSLB在4-6周内确认接受，提供跟踪号码和PIN。注意：超过50%的申请因小错误被退回。
  6. 通过考试 — 通过法律与商业考试和B牌行业考试，均需72%通过。在PSI考试中心进行。未通过需等待至少3周后重考。
  7. 完成石棉考试 — 完成石棉(Asbestos)开卷考试并提交验证表。
  8. 获得保证金和执照 — 获得承包商保证金（最低$25,000）和适用保险，缴纳初始执照费（独资企业$200，非独资企业$350）。
